# Galxe Raffle 🎲

A raffle system powered by SP1 zero-knowledge proofs and Galxe Raffle smart contracts.

## 🏗️ Project Structure

- `zk/`: Contains the SP1 RISC-V program and proof generation scripts.
- `contracts/`: Contains the Galxe Raffle smart contracts.
